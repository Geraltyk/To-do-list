# To-do-list
Приложение представляющее собой список задач.

# Цель проекта
Создать удобный инструмент для планирования и отслеживания выполнения задач с возможностью их организации и редактирования.

# Функционал
1. Создание задачи:
- Название (обязательно).
- Описание (опционально).
- Срок выполнения.
- Приоритет (низкий, средний, высокий).
- Категория.

2. Редактирование задачи:
- Изменение всех атрибутов
- Удаление задачи
- Отметка о выполнении 
- Просмотр всех задач
- Фильтрация по статусу (активные/выполненные)
- Фильтрация по приоритету
- Фильтрация по категории
- Группировка задач по категориям
- Сортировка по дате создания, сроку выполнения, приоритету

# Стек
1. Язык программирования : Java 21
2. Среда разработки : Intellij IDE
3. Интерфейс : Консольное приложение
