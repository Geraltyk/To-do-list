# To-do-list
Простое консольное приложение для управления задачами. Приложение позволяет эффективно организовывать, отслеживать и управлять задачами с помощью интуитивно понятного интерфейса.

## Цель проекта
Создать удобный инструмент для планирования и отслеживания выполнения задач с возможностью их организации, фильтрации и сортировки.

## Возможности
###  Управление задачами
- **Создание задач** с названием, описанием, сроком выполнения, приоритетом и категорией
- **Полное редактирование** всех атрибутов задачи
- **Удаление задач** по ID
- **Отметка о выполнении**

### Просмотр и организация
- **Просмотр всех задач** в удобном формате
- **Фильтрация задач**:
  - По статусу (активные/выполненные)
  - По приоритету (низкий, средний, высокий)
  - По категории
- **Сортировка** по параметрам:
  - По дате создания
  - По сроку выполнения
  - По приоритету
    
### Работа с данными
- **Загрузка задач** из файла при запуске
- **Ручное сохранение/загрузка** в любой момент
